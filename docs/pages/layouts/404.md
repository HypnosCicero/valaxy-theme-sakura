## 404布局 404

一般情况下遇到404页面都会重定向为首页，除非您取消了重定向

```vue
<template>
  <Layout>
    <template #content>
      <div text="center">
        <div text-4xl>
          <div i-ri-alarm-warning-line inline-block />
        </div>
        <RouterView />
        <div>
          <button btn text-sm m="3 t8" @click="router.back()">
            {{ t('button.back') }}
          </button>
        </div>
      </div>
    </template>
  </Layout>
</template>
```
